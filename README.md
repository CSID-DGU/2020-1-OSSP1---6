### ✔ Description

**단 한 번의 사진 촬영을 통한 유통기한 관리 애플리케이션**

### ✔ 기능

* **유통기한이 명시되어 있지 않은 식품 (ex:채소, 과일)에 대한** 적절한 보관 방법과 섭취 가능 기간을 함께 제공
* **유통기한이 명시되어 있는 유제품에 대한** 유통기한 인식 및 보관

### ✔ 개발 도구
